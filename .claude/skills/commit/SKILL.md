## Commit all pending changes

!`git add -A && git commit`

## Instructions

Create a new commit using all pending changes.

For the commit message, please adhere to the conventional commits standard.
If applicable, use the main data model that was affected as scope.
For obvious changes, please omit a detailed description.

Do not include yourself as a co-author.
