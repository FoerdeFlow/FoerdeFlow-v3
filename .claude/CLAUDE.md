# FördeFlow
FördeFlow ist eine Webanwendung, welche die Organisation der Arbeit der verfassten Studierendenschaft unterstützt.

## Technologiestack

- Nuxt
- Drizzle ORM
- KERN UX (https://kern-ux.de)
- Pug
- TypeScript

## Coding-Stil

- Einhaltung der ESLint-Regeln
- Präferiere `computed` über `ref`/`watch`
- In `computed`, nutze die Arrow-Syntax für Funktionen. Kürze das `return`, falls möglich.
